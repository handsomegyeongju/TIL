# Math.method()

- Math.abs(): 절댓값 반환한다.
```js
    Math.abs(-1);       // 1
    Math.abs('-1');     // 1
    Math.abs('');       // 0
    Math.abs([]);       // 0
```
- Math.round(): 소수점 이하를 반올림한 정수를 반환한다. 
```js
    Math.round(1.4);  // 1
    Math.round(1.6);  // 2
    Math.round(-1.4); // -1
    Math.round(-1.6); // -2
```
